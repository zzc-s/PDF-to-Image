# PDF 批量转图片工具



将 PDF 文件的每一页转换为 PNG 或 JPG 图片，支持一次选择多个 PDF 批量处理。



## 环境要求



- Windows 10/11

- Python 3.10 或更高版本



## 快速开始



1. 双击运行 [`run.bat`](run.bat)

2. 首次运行会自动安装依赖

3. 在界面中点击「添加 PDF」选择文件

4. 选择清晰度预设或手动调整 DPI / 格式

5. 点击「开始转换」



转换结果默认保存在项目目录下的 `output/` 文件夹中。图片直接输出到该目录，命名为 `PDF文件名_page_001.png`、`PDF文件名_page_002.png` 等，不再为每个 PDF 创建子文件夹。



## 手动运行


PS:上面“快速开始”是根据自身C盘中的python3.12解释器运行的,如果不是的话建议手动在终端运行
```bash

pip install -r requirements.txt

python app.py

```

## 清晰度设置说明



### 清晰度预设



| 预设 | DPI | 格式 | 适用场景 |

|------|-----|------|---------|

| 快速预览 | 150 | PNG | 体积小、速度快 |

| 标准 | 200 | PNG | 日常屏幕查看 |

| 高清 | 300 | PNG | 打印级清晰度 |

| **OCR 推荐（默认）** | **400** | **PNG** | 文字识别、发票/合同等小字文档 |

| 超清 | 600 | PNG | 极小文字、扫描级精度 |



添加 PDF 后，界面会显示**预计输出像素尺寸**，便于确认清晰度是否满足需求。



### OCR 场景推荐



- 默认使用 **OCR 推荐（400 DPI + PNG）**

- 文字特别小或识别不准时，改用 **超清（600 DPI）**

- **PNG** 为无损格式，文字边缘更清晰，更适合 OCR

- **JPG** 仅建议用于不做文字识别、且对文件体积敏感的场景



### DPI 与像素参考（A4 页面）



| DPI | 约等于像素 |

|-----|-----------|

| 150 | 1240 × 1754 |

| 200 | 1654 × 2339 |

| 300 | 2480 × 3508 |

| 400 | 3307 × 4678 |

| 600 | 4961 × 7016 |



实际像素以界面「预计输出」显示为准，不同 PDF 页面尺寸会有差异。
